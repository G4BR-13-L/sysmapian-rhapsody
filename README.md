#### Como executar

##### 1. Clone este repositório

##### 2. Acesse esta pasta pelo terminal
```bash 
    cd teste-front-end
```
##### 3. instale as dependencias com npm
> Utilize a versão LTS do Node
```bash
npm install
```

##### 4. Execute com o script do npm 
```bash
npm run dev
```

##### 5. Acesse a aplicação pelo navegador
```bash
localhost:3000
```
